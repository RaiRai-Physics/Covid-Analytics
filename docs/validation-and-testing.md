# Validation and Testing

## Silver Validation
Checks performed:
- Bronze files successfully read
- Silver Parquet outputs written to S3
- no null values in key columns (`full_date`, `state_code`)
- numeric fields cast correctly

## Gold Validation
Checks performed:
- Gold Parquet outputs written successfully
- dimensions and facts generated
- `state_code` retained inside fact Parquet records
- row counts generated by state and date

## Snowflake Validation
Queries executed:
- count rows in each curated table
- verify no null `state_code` in fact tables
- verify min and max `date_id`
- verify distribution of rows by `state_code`

## Final Validation Queries
Examples:
- latest available date
- row counts by state
- positivity trend by state
- top states by new cases on latest available date
